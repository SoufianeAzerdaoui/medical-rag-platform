# Sample Data

This folder is reserved for future non-sensitive project assets used during experimentation and evaluation.

Store only small, reviewable files such as:

- sample medical laboratory PDFs that you are allowed to use,
- extracted JSON outputs generated from those PDFs,
- evaluation queries and expected evidence notes,
- anonymized artifacts for testing extraction or retrieval behavior.

Do not commit:

- binary dumps that are too large to review,
- confidential patient data,
- uncontrolled datasets copied from external systems.

Keep this folder lightweight and suitable for academic supervision.
